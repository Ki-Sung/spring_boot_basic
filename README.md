## 프로젝트 실행 방법
- mvnw 방식으로 실행
  - TestProject Working Directory로 가서 아래의 명령어 입력 
  ```
  ./mvnw spring-boot:run
  ```
- Intelli J IDEA 환경에서 실행
  - IDEA에 Working Directory로 이동
  - src/main/com/example/TestProject로 이동
  - "TestProjectApplication.java" 파일 실행
